# bookstore_app
Backend - **sqlite3** database code. Interact with a database.

Frontend - the code of **Tkinter** library.

A program that stores this books information:

- Title
- Author
- Year
- ISBN

Users can:

- View all records
- Search an entry
- Add an entry
- Update an entry
- Delete
- Close
